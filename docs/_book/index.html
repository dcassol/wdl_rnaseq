Welcome

About

This repository containers RNA-seq pipeline, docker container and sample data for testing. Pipelines are written in WDL language and Cromwell workflow execution engine is used to run them.

Is this workflow useful for you?

Expected output

  • Fastqc Quality Report
  • Reads Trimming with Trimmomatic
  • Align Reads to Reference Genome using HISAT2 (BAM files)
  • Expression quantification using summarizeOverlaps (GenomicFeatures) (Lawrence et al. 2013)
  • Differential Expression Matrix using DESeq2
  • Create Feature set and Filtered Differential Expression Matrix

Quick Start - No Installation required 

docker run  -dP dcassol/wdl_rnaseq
docker ps 
docker exec -it <IMAGE NAME> /bin/bash
cromwell run rnaseq.wdl -i rnaseq.json -o rnaseq.options.json

Now, you can test and edit the pipeline for your data. For more details, please check the Sample Data section. Have fun!

Quick Start

Step 1: Clone the git repository to a directory on your computer:

https://github.com/dcassol/wdl_rnaseq.git

Note: Ensure you have the correct permissions to run the pipeline - here we are using Docker. A further section will be provided containing more details if you are running in an HPC environment.

Step 2: To run the pipeline use the command below:

cromwell run rnaseq.wdl -i rnaseq.json -o rnaseq.options.json 
## OR
java -jar cromwell-72.jar run rnaseq.wdl -i rnaseq.json -o rnaseq.options.json

Please, make sure use the correct PATH to cromwell-72.jar file.
